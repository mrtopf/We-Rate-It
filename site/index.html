We Rate It

Ein kollaboratives Altersbewertungssystem für Web-Sites

Was ist es?

WeRateIt ist ein Experiment, das aus der Diskussion rund um die gescheierte Novellierung des Jugendmedienschutz-Staatsvertrags (JMStV) entstanden ist.

Die Grundidee ist, dass nicht Webseiten-Betreiber eine Altersklassifizierung vornehmen, sondern die Besucher der Websites. Dies soll kollaborativ geschehen, d.h. dass eine Vielzahl von Bewertungen in die endgültige Bewertung einfliessen sollen. Im Moment ist es allerdings noch ein Experiment, was bedeutet, dass nur die rudimentären Features implementiert sind.

Was geht und was geht nicht?

Die Bewertung selbst geschieht durch ein Browser-Plugin, welches im Moment nur für Google Chrome vorliegt. Eine Filterung von Webseiten auf Basis dieser Bewertungen ist im Moment noch nicht implementiert

Bewertungen werden gespeichert, es gibt aber keine Garantie, dass nicht noch einmal von vorne begonnen werden muss, auch wenn wir versuchen werden, die schon abgegebenen Bewertungen zu behalten.

Bedienung

Datennutzungserklärung

Im Moment werden folgende Daten gespeichert:
  • Eine User-ID wird bei Installation des Plugins automatisch registriert und im Plugin wie in der Datenbank gespeichert. Bei einer Neuinstallation wird allerdings eine neue ID vergeben, ebenso auf unterschiedlichen Browsern.
  • Pro Rating werden User-ID, Rating (0-18), Domain und Zeitstempel gespeichert. Eine Zuordnung der bewerteten Sites (aber nur der Domain) zu einer User-ID ist demnach möglich. Eine Zuordnung wird allerdings nur bei einer Bewertung gespeichert, bei einem reinen Besuch ohne Bewertung werden keine Daten vom Browser an die Datenbank übertragen.
